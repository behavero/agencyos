# Get subscribers

GET https://api.fanvue.com/subscribers

Returns a paginated list of users who have subscriptions to the authenticated user.

    <Info>Scope required: `read:fan`</Info>

Reference: https://api.fanvue.com/docs/api-reference/reference/creators/list-subscribers
